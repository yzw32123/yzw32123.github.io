领达-为师

温故而知新，可以为师矣。

师资力量

为师公众号发展以来，以务实并开放的态度在线下和线上寻找资历深厚的教师，有近百人的师资团队在背后默默付出着力量，贡献了许许多多优质的课程！我们欢迎更多优秀的老师加入我们，一起创造一个知识无界限的美好世界！

用户人群

现在我们的用户群体多是一群渴望知识的学生，在未来我们希望任何一个渴望得到知识的个体，都能免费从‘为师’获得想要学习的知识或技能！

线上生态

在‘为师’公众号内，您可以免费并系统的学习，并且支持课件下载功能。如果您线下的老师已经是‘为师’的用户，那么可以享受特定课程内容的推送，方便您在课后预习和复习课堂内容。
